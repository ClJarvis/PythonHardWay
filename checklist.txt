A checklist for functions.

1) Did start with function defintion? def
2) Does function name  contain only characters and underscores '_' ?
3) Is there an open parenthesis after the function Name? '('
4) Did you put agruments after open parenthesis seperated by commas ','?
5) Did you make each argument unique?
6) Did you close parenthesis and add colon? '):'
7) Did you indent all code in function FOUR SPACES?
8) Did you end the function by writng with no indent?
